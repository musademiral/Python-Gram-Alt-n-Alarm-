Canlı Altın Fiyatı Takip ve Alarm Sistemi (Python)

Bu proje, Truncgil Finance API kullanarak güncel gram altın fiyatlarını çeken ve kullanıcının belirlediği hedef fiyata ulaşıldığında Windows Masaüstü Bildirimi (Pop-up) gönderen bir otomasyon scriptidir.

Kurulum:

Projeyi bilgisayarınıza indirdikten sonra gerekli kütüphaneleri kurmak için terminale şu komutu yazın:
`pip install -r requirements.txt`

## 💻 Kullanım
Programı çalıştırın:
`python main.py`
Ekranda çıkan "Lütfen Altının Altın Alarmı İçin Sınır Ekleyiniz: " sorusuna hedef fiyatınızı (örn: 7300) yazın ve arkanıza yaslanın. Fiyat o seviyeyi aştığında sağ alttan bildirim gelecektir!
